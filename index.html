🍣 육회바른연어 YouTube 모니터링

T1 직접 경쟁 + T2 소비상황 + T3 콘텐츠 벤치마크
데이터 로딩 중...
육회바른연어의 YouTube 경쟁 환경은 단순 외식 브랜드 비교가 아니라, '육회·연어·초밥·덮밥·배달·혼밥·안주' 수요를 둘러싼 콘텐츠 점유 경쟁으로 봐야 합니다. 따라서 본 대시보드는 직접 경쟁 브랜드(T1), 소비상황 경쟁 키워드(T2), 콘텐츠 벤치마크 채널(T3)을 분리해 분석합니다.

T1 직접 경쟁 기준 우리 위치

동일 카테고리(육회·연어·초밥·덮밥) 브랜드 6개 중 자사 SOV 순위와 핵심 지표

최근 이슈 영상 (자사)

부정률 또는 buzz score 기준 상위 영상

기회 키워드

T2 소비상황 중 자사 노출이 약한 영역
T1 직접 경쟁자는 육회바른연어와 동일하게 육회, 연어, 초밥, 덮밥, 세트 메뉴 수요를 공략하는 브랜드입니다. 이 그룹은 Earned SOV, Engagement SOV, 부정률을 기준으로 자사와 직접 비교합니다.

T1 브랜드 비교 표

6개 직접 경쟁 브랜드의 전체 지표 비교 (자사 = 하이라이트)

Earned SOV (조회수 점유율)

Engagement SOV (좋아요+댓글 점유율)

T2 소비상황 경쟁자는 브랜드 정체성은 다르지만, 소비자가 같은 식사·안주·야식·배달 상황에서 대체 선택할 수 있는 브랜드와 키워드입니다. 이 그룹은 브랜드 순위보다 키워드별 콘텐츠 기회 분석에 활용합니다.

소비상황별 기회 분석

6개 소비상황: 육회 배달 / 연어덮밥 / 육회초밥·연어초밥 / 가성비 술안주 / 야식·배달 / 혼밥·점심
T3 콘텐츠 벤치마크는 경쟁 브랜드가 아니라, 유튜브에서 음식 콘텐츠가 확산되는 방식과 포맷을 학습하기 위한 참고군입니다. 이 그룹은 SOV 경쟁 순위에 포함하지 않고, 제목·썸네일·쇼츠 포맷·댓글 반응 분석에 활용합니다.

벤치마크 채널/포맷

또간집, 성시경 먹을텐데, 맛상무, 회사랑, 푸디랜드, 야미보이, 푸드킹덤, 모두의 프랜차이즈, 창업 인터뷰 채널, 지역 맛집 쇼츠 채널
자사(육회바른연어) 관련 영상만 깊게 분석합니다. 영상명, 채널, 영상 유형, 조회수, 좋아요, 댓글수, ER, 긍·부정 댓글, 언급 메뉴, 언급 매장, 개선 포인트를 정리합니다.

전체 영상 (최근순)

브랜드명이 아닌 소비자 검색 키워드 중심의 분석입니다. 키워드별 조회수, 영상수, 쇼츠 비중, 긍·부정 반응, 자사 노출 여부를 확인합니다.

키워드별 콘텐츠 현황

육회바른연어, 육바연, 육회 배달, 연어덮밥, 육회초밥, 연어초밥, 육회 연어 세트, 가성비 안주, 야식 추천, 혼밥 추천, 배달 안주
본 대시보드는 육회바른연어의 YouTube 경쟁 환경을 단순 브랜드 비교가 아니라 소비자가 실제로 대체 선택할 수 있는 메뉴·상황 기준으로 재정의합니다. T1은 육회·연어·초밥·덮밥 카테고리에서 직접 비교되는 브랜드이며, T2는 술자리·혼밥·야식·배달 상황에서 같은 소비 예산을 두고 경쟁하는 브랜드/키워드입니다. T3는 경쟁자가 아니라 유튜브 콘텐츠 형식과 확산 방식을 벤치마킹하기 위한 채널/포맷입니다.

용어 설명

Earned SOV

각 브랜드 영상의 총 조회수 점유율. T1 6개 브랜드 합계 대비 비중. 소비자가 자발적으로 본 콘텐츠 점유율을 나타냅니다.

Engagement SOV

좋아요+댓글 합계의 점유율. 조회수보다 적극적 반응의 점유율을 보여줍니다.

Owned / Sponsored SOV

Owned = 자사 채널 업로드 / Sponsored = 협찬·광고로 추정되는 외부 채널 / Earned = 비협찬 자발 콘텐츠.

ER (Engagement Rate)

(좋아요 + 댓글) / 조회수. 영상 단위 반응의 질을 측정합니다. 5% 이상이면 매우 높음.

부정률

댓글 중 부정 감성으로 분류된 비율. 10% 초과 시 경고로 표시됩니다.

Buzz Score

최근 화제성 점수 (조회수 + ER + 신선도 가중). 100 만점.

Creator Tier

구독자 수 기준: Nano(<10K), Micro(10K–100K), Mid(100K–500K), Macro(500K–1M), Mega(1M+), Official(브랜드 공식).

쇼츠 비중

60초 이하 쇼츠 영상 수 / 전체 영상 수. 쇼츠 중심 콘텐츠 vs 롱폼 비율.
